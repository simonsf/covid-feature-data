# covid-feature-data
Features extracted from CT images

----------------
Data descriptions
----------------
Total there are 2685 subjects, including 1658 COVID-19 patients and 1027 CAP patients. 
The task is to differentiate COVID-19 from CAP patients.

Please refer to the below paper for the detailed description of data. 
Shi, F., Xia, L., Shan, F., Song, B., Wu, D., Wei, Y., Yuan, H., Jiang, H., He, Y., Gao, Y., Sui, H., & Shen, D. (2021). Large-scale screening of COVID-19 from community acquired pneumonia using infection size-aware classification.Â Physics in medicine and biology, 10.1088/1361-6560/abe838. Advance online publication. https://doi.org/10.1088/1361-6560/abe838	

----------------
Feature descriptions	
----------------
Features were extracted from CT images, including handcrafted and Radiomics features.

Column	Feature

F-CT	        Radiomics

CU-DX	        Histogram

DY-EV	        Number

EW-EX	        Mean and std of intensity of infections

EY-FA, GI-GL	Surface

FB-GH	        Volume
